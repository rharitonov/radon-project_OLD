Папка основного расширения на проекте внедрения BC. 
Все *.al файлы должны храниться тут.
Основное правило: один объект (расширяемый стандартный или кастомный) - один файл *.al.
На примере, это может быть так что, если в ходе проекта необходимо добавить два экстеншена, то они добавляются в один файл *.al
